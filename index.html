Divvy
Divvy is a team based project management web application. Making use of websockets and Pusher.com the application is realized in real time. Meaning whenever a user performs any action within a project, all other users within that project see that action immediately.
technologies used: Laravel 5, Pusher, jQuery, Codeception, Bootstrap, Sass
Biltmore Print
The client was transitioning from a franchise to an independent business, they needed a website that provided the same functionality as their former franchise site. technologies used: Laravel, jQuery, Bourbon + Neat
MVC Framework
Before learning a PHP framework I wanted to understand the problems they were designed to solve, so I built my own as a learning exercise. The finished product is fully object oriented and features common framework constructs such as: URL routing, object relational mapping, and request/response objects. I used the framework to build a simple content management system that allows admin users to create, edit and delete content.
technologies used: PHP, MySql, Composer, Bootstrap, Bower, Grunt
